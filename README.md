# Nombre del Proyecto

## Descripción

Back Sakacancha

## Instalación

1. Clonar el repositorio
2. Instalar las dependencias

```bash
npm install
```

3. Crear el archivo .env y agregar las variables de entorno

4. Generar el prisma client

```bash
npm run generate
```

5. Iniciar el servidor

```bash
npm start:server
```
